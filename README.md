# Projeto de Rede Industrial - Cisco Packet Tracer

Este projeto simula a infraestrutura de rede de uma fábrica, focando na comunicação entre setores e servidores de serviços.

## Tecnologias Utilizadas
- **Cisco Packet Tracer**
- **Protocolos:** DHCP, DNS.
- **Hardware:** Router 1841, Switches 2950-24, Servidores e PCs.

## Funcionalidades
- **Servidor DNS:** Tradução do domínio `www.fabrica.com` para o IP do servidor.
- **DHCP:** Atribuição dinâmica de IPs para as máquinas das áreas de Manufatura e Acabamento.
- **Gateway:** Configuração de rota de saída para comunicação entre diferentes sub-redes.
